# 🚌 BRTS Bus Management System (Java)

This is my Semester 1 Java project based on Ahmedabad BRTS system.  
It is a console-based application that allows users to register, login, and book bus tickets.

---

## 📌 Features
- 👤 User Registration & Login system
- 🎫 Ticket Booking
- 🔄 Ticket Update & Cancel
- 💸 Fare Calculation with discounts:
  - Student (40%)
  - Senior Citizen (60%)
- 🚌 Bus Route & Station display
- 📄 Passenger Information system

---

## 🛠 Technologies Used
- Java
- OOP Concepts (Classes, Objects, Inheritance)
- Arrays
- Scanner (User Input)

---

## ▶️ How to Run

1. Compile the program: javac Run.java
2. Run the program: java Run


---

## 💡 Project Description

This system simulates a basic BRTS (Bus Rapid Transit System) where users can:
- Register and login
- View available bus routes
- Book tickets between stations
- Calculate fare automatically
- Manage their bookings
